Mini Projects

Cash Flow Optimization via Linear Programming

Formulated a multi-period investment and loan optimization problem as a Linear Program. Solved it using the Revised Simplex Method and validated the results using numerical solvers to achieve optimal fund allocation and cash flow stability.

Coal Plant Efficiency Optimization

Modeled thermal power plant efficiency as a quadratic optimization problem. Trained using Gradient Descent, Mini-batch GD, and Adam Optimizer, and compared solutions from Newton’s Method, BFGS, and Conjugate Gradient.

Indian Name Generation and Transliteration using FNN & RNN

Developed Feedforward (FNN), Recurrent (RNN), and Attention-based RNN models for Indian name generation and transliteration. Implemented Beam Search for decoding. Achieved BLEU scores of 0.70 (RNN) and 0.789 (RNN with Attention).

Singular Value Decomposition (SVD)

Demonstrated image compression using SVD based on the Eckart–Young theorem, balancing reconstruction accuracy and compression ratio.

PageRank Algorithm

Implemented PageRank using the Power Iteration method to simulate web link structures. Verified convergence properties and rank stability across iterations.
